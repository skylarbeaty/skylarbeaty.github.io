Projects

Phase Drift

An arcade shooter, designed around mobile friendly controls, where you play on new stages every run. It was released on the App Store and Google Play with integration of services on both platforms. This project has been maintained on the Play Store for multiple years.

The procedural levels work by intelligently pasting pixel textures on top of eachother, and using that to spawn the levels enemies. The Implementation of the user interface makes use of Unity's scriptable objects in order to work with the procedurally generated shops and levels.

Procedural Generation

I’ve created several different methods of procedurally generating levels, including custom solutions for my own projects, like I did for Phase Drift. I’ve also recreated methods from popular games, like Spelunky, The Binding of Isaac, and Nuclear Throne.

I continually come back to procedural generation as an area of study, exploring different ways it can be done. Nuclear Throne uses a random walker (pictured), while Spelunky and Isaac rely on mixing templates with rules for how “rooms” connect. Phase Drift has a template system that was made specially for it. There are areas of proc gen I have been studying that I hope to implement at some point, like Wave Function Collapse or Hybrid Systems with both procedural and hand craft elements. I have a special interest in roles that allow me to explore this niche.
